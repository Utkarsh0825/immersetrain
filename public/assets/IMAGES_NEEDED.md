# Images Needed for ImmerseTrain

Place images in these folders. Use PNG with transparent backgrounds where noted.

## /public/assets/photos/
- `hero-headset.png` — High-quality VR headset photo (transparent bg) — used in hero section as alternative to 3D model
- `training-scene.jpg` — 360° training scenario screenshot — used in social proof section
- `dashboard-preview.png` — Dashboard UI screenshot (transparent bg) — used in features section

## /public/assets/icons/
- SVG icons are generated in code — no files needed here unless you want custom ones

## Optional Enhancements
- `360-demo.gif` — Animated GIF showing 360° video rotation — can be used on demo page
- `team-photo.jpg` — Team or workplace photo for the about section
